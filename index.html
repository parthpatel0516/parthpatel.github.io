Portfolio

Supa Nova

Supa nova is easy to develop individual training plans for your employees on an online platform that seamlessly integrates with the tasks that need to be performed in order to best serve your clients. In this project we created the same functionality as Trello to manage the projects and tasks. Project owners can create new tasks and assign them to multiple users.

Technologies used: Laravel, AngularJs, AJAX, jQuery

Umbrella

It is related to financial service providers. It provides health and funeral plans. You can manage beneficiaries, Childs, spouses and individuals to plans. It uses Xero for managing the invoicing, bank reconciliation for the payment of the purchased plans. It uses Paym8 payment gateway for payments.

Technologies used: Laravel, HTML, CSS, jQuery, AJAX

Cabbyx

Cabbyx is an app that helps customers to find the service providers nearby using geo-location. Service providers can create services and users can select any services and contact service providers or they can chat. The backend is created in laravel to manage all the users and other modules.

Technologies used: Laravel, Ionic, HTML, Bootstrap, CSS, Javascript, jQuery, Ajax

Planet supplements

Planet supplements is popular for selling a wide range of high-quality fitness products that people need to enrich their muscles and body. In the backend super admin can manage all the users, products, categories, orders, and other modules. In this project the frontend is in angular and the backend is in laravel.

Technologies used: Laravel, Angular, HTML, Bootstrap, CSS, Javascript, jQuery, Ajax

Eatcard

Eatcard is the biggest product I have ever worked on. It is an automatic system for every type of catering industry that meets store needs. It includes the functionalities like online reservation, real time planner, dine-in, takeaway, kiosk, QR table scanning, all you can eat and many third party API integration.

Technologies used: Laravel, Node JS and Socket.io(realtime updates and notifications), Redis, Ionic, HTML, Bootstrap, CSS, Javascript, jQuery, Ajax

E-rehab

E-rehab is the physical therapy websites and online marketing services for physical therapy private practices. It helps user to get new patients and provide services like newsletter, survey systems, google reviews, etc.

Technologies used: Laravel, Livewire, Alpine js, HTML, Bootstrap, CSS, Javascript, jQuery, Ajax
